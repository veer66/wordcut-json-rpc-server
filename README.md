# wordcut-json-rpc-server

## How to run

```
docker run -d --name wordcut --net=host veer66/wordcut-json-rpc-server 
```

## How to build and push

```
docker build -t veer66/wordcut-json-rpc-server .
docker push veer66/wordcut-json-rpc-server
```
